# Laravel API & Vue SPA: Example App  

## Stack  
- PHP 8.2
- MariaDB 10
- Node 18
- Docker
- Laravel 10
- Octane Roadrunner
- Telescope
- Vue 3
- Vite
- Yarn
- Tailwind CSS

### Build with docker  
#### Don't use this in a real application, just for a quick demo 
```
docker-compose up -d
```
Go to http://localhost:8000
